Motivation
Why Real-Time Operator Training?
Modern grids integrate converter-based resources with fast, non-linear dynamics. Traditional training relies on static scenarios and proprietary, closed-source platforms. Operators face cascading failures, renewable uncertainty, and multi-layer control that demand immersive, real-time simulation.
Core Challenge
Simulate a 6,000+ bus transmission network faster than real-time on commodity hardware, within the 1–2 s SCADA refresh constraint, while retaining full operational fidelity for breaker-level interactions.

New Work — Extension 2026
Adaptive Model Selection (AMS) — AI-Driven Fidelity Control
The AMS module predicts component-level dynamic activity before each training session, selecting only the fidelity needed for the active scenario. A Graph Attention Network (GATv2) learns which network components participate actively in disturbance propagation, enabling hybrid models that retain full detail where needed and simplify elsewhere.
